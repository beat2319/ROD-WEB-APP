### Current Comments:
  - "_detections_ is generally a term reserved for a situation where there is a confirmed case, supported by laboratory testing. At least how we've been using the term. You're going to pick up all sorts of changes that have nothing to do with ROD, so you need to keep that in mind. Also, should cite the vegetation layer you're using and I'd be careful about red-green color schemes, there are plenty of colorblind folks." - Professor Ryan Perroy
